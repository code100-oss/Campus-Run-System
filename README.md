# 二手商店平台 ## 1. 项目概述 本项目旨在搭建一个面向校园/社区用户的二手商品交易平台，解决闲置物品流转效率低、信息不对称的问题，为用户提供安全、便捷、可信赖的二手商品买卖服务。 目标用户群体为在校大学生、社区居民等有闲置物品出售或二手好物采购需求的个人用户，覆盖数码、服饰、书籍、生活用品等全品类二手交易场景。 ## 2. 功能介绍（完整5大核心模块） ### 2.1 用户管理 为平台用户提供全生命周期的账号服务，包含用户注册、账号登录、个人信息维护（头像、昵称、收货地址、实名认证）、账号安全设置（密码修改、登录保护）等核心能力，保障用户账号安全与信息管理需求。 ### 2.2 商品管理 二手商品的核心运营模块，支持卖家发布商品（上传图片、填写商品描述、定价、分类标签）、编辑/下架已发布商品、管理商品订单；同时支持买家按分类、价格、地区、新旧程度等条件筛选商品，查看商品详情、收藏心仪商品，实现商品信息的高效管理与检索。 ### 2.3 交易管理 保障交易流程闭环的核心模块，包含订单创建、订单支付、订单发货、订单确认收货、售后申请（退货退款、换货）、交易评价等全流程功能，同时集成交易担保机制，保障买卖双方的资金与权益安全，规范交易行为。 ### 2.4 社交与互动 提升平台活跃度的服务模块，支持用户关注感兴趣的卖家、私信沟通商品细节、发布商品咨询、分享好物到社交圈、查看他人交易评价，搭建用户间的信任桥梁，增强平台的社区属性。 ### 2.5 数据统计与运营管理 面向平台管理员与卖家的运营模块，支持卖家查看店铺数据（商品浏览量、订单量、成交额、用户画像）；支持管理员进行平台数据统计（交易总额、用户增长、商品分类占比）、违规商品审核、用户行为管理、平台规则配置，保障平台合规、高效运营。 ## 3. 关键说明（可选补充，提升专业性） ### 3.1 技术栈说明 - 前端：Vue.js / React + Element UI / Ant Design - 后端：Spring Boot / Django + MySQL - 部署：Docker + Nginx - 其他：微信/支付宝支付接口、图片存储服务 ### 3.2 项目特色 - 实名认证+交易担保，保障交易安全 - 多维度商品筛选，提升找货效率 - 社区化互动，增强用户粘性 - 轻量化运营，适配校园/社区场景 ### 3.3 后续迭代计划 - 新增二手租赁功能 - 优化AI商品智能定价 - 拓展同城自提服务[这里输入链接描述](https://github.com/code100-oss/Campus-Run-System.git)
