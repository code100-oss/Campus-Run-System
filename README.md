# Campus-Run-System
1. 用户模块 注册、登录、个人信息管理（昵称、头像、联系方式） 实名认证（可选）、账号安全设置 2. 商品模块 商品发布：上传图片、填写标题 / 描述 / 价格 / 分类 / 成色 商品管理：编辑、下架、删除、上下架状态切换 商品浏览：分类筛选、搜索、热门推荐、新品展示 3. 交易沟通模块 买家与卖家在线聊天（私信） 收藏商品、浏览历史记录 4. 订单与交易 下单、订单状态（待付款 / 待发货 / 待收货 / 已完成 / 已取消） 简易支付对接（模拟 / 第三方支付） 评价功能（买家评价卖家，商品互评） 5. 后台管理（管理员） 用户管理、商品审核、违规处理、订单管理、数据统计
